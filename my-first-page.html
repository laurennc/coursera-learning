The img tag works as an in-line element.

Using width and height tells the browser how much space to save for the image and how to partition things on the screen. That way the screen/text/other elements don't jump around as the page loads the new elements. Also, if the URL is broken, it won't destroy the whole layout of your page, particularly if the layout relied on the size of the image.
